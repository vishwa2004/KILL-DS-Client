#  KILL Client — Testing Version 1.0

A custom development client for **Deadshot.io** with enhanced features and mod support.

## Features

- ✅ No Ads
- ✅ Lobby chat
- ✅ Clan chat
- ✅ Ingame VC
- ✅ Unlimited fps
- ✅ New profile section
- ✅ New AWP Scope
- ✅ KILL gun skin
- ✅ Sniper mode
- ✅ Leaderboard rank search
- ✅ Headshot sound
- ✅ Key layout
- ✅ Fps display
- ✅ 4 crosshair options

---

##  Installation

Follow the steps below to set up the project:

```bash
git clone https://github.com/vishwa2004/KILL-DS-Client
npm install
cd KILL-DS-Client
npm start
```
